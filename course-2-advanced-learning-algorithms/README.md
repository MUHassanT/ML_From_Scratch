# Neural Networks: Forward Propagation From Scratch

This notebook is part of my Machine Learning from Scratch journey, based on Course 2 of Andrew Ng's Machine Learning Specialization.

The goal of this notebook is to understand how a basic neural network makes predictions using forward propagation. Instead of focusing on training or backpropagation, this notebook focuses on the structure of a neural network and how data moves through its layers.

## What I Implemented

In this notebook, I implemented:

* A sigmoid activation function
* A single neuron from scratch
* A dense layer using a loop-based approach
* Forward propagation through a small neural network
* A vectorized dense layer using NumPy
* A simple TensorFlow/Keras comparison

## Main Concepts

This notebook focuses on the following ideas:

* A neuron as an extension of logistic regression
* Weights, biases, and activations
* How multiple neurons form a dense layer
* Why a weight matrix is two-dimensional
* How outputs from one layer become inputs to the next
* The difference between loop-based and vectorized implementations

## Tools Used

* Python
* NumPy
* Matplotlib
* TensorFlow/Keras

## Note

This notebook only covers forward propagation. Training, loss functions, gradient descent, and backpropagation will be covered in later notebooks.
