# ML From Scratch

This repository documents my journey of learning Machine Learning from scratch.

I am currently learning through Andrew Ng's Machine Learning courses and implementing the core algorithms myself to understand how they work internally, instead of only using built-in libraries.

The goal of this repository is to build a clear record of the topics I study, the code I write, and the concepts I understand along the way.
