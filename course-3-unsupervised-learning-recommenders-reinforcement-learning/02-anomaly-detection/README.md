# Gaussian Anomaly Detection from Scratch

This notebook implements a Gaussian-based anomaly detection algorithm from scratch using NumPy and SciPy. It explores the statistical foundations of anomaly detection, implements the core algorithm, and evaluates its performance on a synthetic dataset.

## Topics Covered

- Introduction to Anomaly Detection
- Gaussian Distribution
- Maximum Likelihood Estimation (MLE)
- Gaussian Parameter Estimation
- Probability Density Estimation
- Threshold Selection
- Precision, Recall, and F1 Score
- Experimental Analysis
- Limitations of Gaussian Models

## Features

- Estimate Gaussian parameters from data
- Compute multivariate Gaussian probability densities
- Automatically select the optimal threshold using F1 Score
- Detect anomalous observations
- Visualize Gaussian contours and feature distributions
- Analyze the effect of different threshold values
